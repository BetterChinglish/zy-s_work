层叠性
后来居上，如果样式存在冲突，则离的远的会被覆盖
继承性
子标签会继承父标签的某些样式，如文本颜色字号，比如div标签里面的p标签
恰当地使用可以简化代码
行高的继承
12px为字体大小，1.5则为行高是字体大小的1.5倍（可以不更单位，也可以直接写具体行高数值）
利用继承性
body里面的标签文本都会有字体大小1.5倍的行高
优先级
当同一个元素指定多个选择器，就会产生优先级
如果选择器相同，则执行层叠性
如果选择器不同，则按权重执行
选择器权重：！important>行内样式style>ID选择器>类选择器，伪类选择器>元素选择器>继承或*
如上标注
权重叠加，出现在有复合选择器时，需要计算权重，权重叠加不会有进位
分类描述的越细致，优先级越高，例如ul>li要大于li
