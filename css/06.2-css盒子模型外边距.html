margin属性用于控制外边框，即盒子和盒子间的距离 margin-方位名词：px;
恰当地使用可以合并外边框 "即，尝试尽量只给一个盒子添加margin值
外边距可以让块级盒子水平居中，但必须满足两个条件：
1·盒子必须指定宽度
2·盒子左右外边距都设置为auto
3·以上是块级元素
4·行内或行内块元素居中添加text-ailgn:center即可
外边距合并
使用margin定义块元素的垂直外边距时，可能会出现外边距的合并
对于两个嵌套（父子）关系的块元素，父元素有上边距同时子元素也有上边距，此时父元素会塌陷较大外边距值
解决方法
1·为父元素定义上边框
2·为父元素定义上内边框
3·为父元素添加overflow:hidden
清除内外边框
很多浏览器都带有默认的边距值，在布局之前要清除掉
*{ badding：0（清除内边距） margin：0（清除外边距） }
