背景属性可以设置 背景颜色，背景图片，背景平铺，背景图片位置，背景图像固定等
一般情况下背景颜色默认值为透明transparent
背景图片background-image实际开发用于logo或小图标装饰，便于控制位置
背景平铺使用background-repeat属性,默认情况下都是平铺
改变图片在背景中的位置使用background-position，其中参数代表坐标，可以使用方位名词和精确单位z
如果使用的是方位名词（top，center，bottom，left，right），则方位名词的先后顺序不影响表示
如果只使用了一个方位名词，则系统默认居中显示
如果使用精确单位，则前者表示x，后者表示y，若只填写一项则表示x，另一项默认垂直居中(css3会接触z轴)
混合单位及二者的结合使用，是其中的x或y参数被方位名词代替
background-attachment属性设置背景图像是否固定或者随着页面的其余部分滚动（后期可以制作视差滚动效果）
背景里面，无顺序要求
background：背景颜色，背景图片地址，背景平铺，背景图像滚动，背景图片位置
由css3提供,最后一个参数为alpha（透明度），取值范围在0~1之间
